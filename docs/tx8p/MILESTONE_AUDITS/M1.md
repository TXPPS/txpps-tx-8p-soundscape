# M1 — Chassis + Tokens + LCD Shell + Init Preset

## What was built
- Design tokens in `src/styles.css` (oklch): chassis, panel-recess, engraving, LCD greens, amber, red, LEDs, knob palette, spacing scale, motion timings, shadows. `engraved`, `engraved-dim`, `panel-recessed`, `chassis-surface` utilities. JetBrains Mono `@font-face` + preload.
- Chassis primitives: `Chassis`, `Panel`, `EngravedLabel`, `Badge`.
- Controls: `Knob` (SVG-like rotary, drag/wheel/dblclick/keys/ARIA), `LedButton`, `IconButton`, `Wheel` (pitch with spring, mod).
- LCD: `PresetLCD` with dot-matrix scanline overlay, glow, 2×20 char grid, mode-driven state machine: `boot → preset`, `param`, `panic`, `message`, `loading`. Auto-releases parameter feedback after 900ms.
- Layout: TopBar (Badge · LCD · Quick Access · Settings), panel tab strip, PerfStrip (pitch, mod, octave, hold, poly/mono/uni, panic), placeholder InstrumentPanels for all six panel groups, Keyboard (multitouch pointer-authoritative, no audio yet).
- State stores (zustand): `uiStore` (active panel, LCD mode, flash-with-ttl), `presetStore`, `perfStore`.
- Head metadata updated (`__root.tsx` + `/` route): TX-8P title, description, og:*, twitter:card; theme-color; font preload.
